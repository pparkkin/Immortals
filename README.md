Immortals
=========

Akka experiment with networking and game of life.

Run It
------

Use Maven to compile and install to local repo.

```
$ cd Immortals
$ mvn clean install
```

Start a server.

```
$ cd ImmortalsServer
$ mvn scala:run
```

Start a client.

```
$ cd ImmortalsClient
$ mvn scala:run
```

Type in "localhost" in the dialog that opens, and press "connect".
